# Agent Structure (논리적 트리)

- 모카: 총괄 비서 (단일 활성 에이전트)
  - 앱개발 전문가: 앱 설계/개발/배포/자동화/품질 지원

## 호출 규칙(수동 호출)
- 기본: 모카
- 앱개발 작업 시 모카가 `app-dev` 서브에이전트에 위임 가능
  (현재는 자동 바인딩 미적용, 필요 요청 시 수동 전달).

> OpenClaw의 기본 라우팅은 현재 채널/계정/peer 단위입니다.
> 키워드 수동 호출 규칙은 운영 환경에 따라 적용됩니다.

## 실행 규칙
- 독립적인 작업은 병행 수행한다.
- 실시간 로컬 작업(메모/리스트/설정)은 즉시 처리하고,
  네트워크형 정보검색은 필요 시 비동기 실행해 모카가 통합 집계한다.
