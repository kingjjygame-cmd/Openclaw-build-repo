## 호출 규칙(수동 호출)
- 기본: 모카
- 명시 호출 우선:
  - "여행플래너" 또는 "여행전문가"
  - "경제전문가"
  - "한국경제 전문가"
  - "부동산 전문가"
  - "세계경제 전문가"
  - "한국주식 전문가"
  - "미국주식 전문가"
  - "정보검색 전문가"
  - "기술 전문가"
  - "오픈클로 비서"

> OpenClaw의 기본 라우팅은 현재 채널/계정/peer 단위이며
> 현재는 키워드 자동 라우팅 대신 수동 호출 형태로 운영합니다.
>
- 실행 규칙
  - 독립적인 작업은 병행 수행한다.
  - 실시간 로컬 작업(메모/리스트/설정)은 즉시 처리하고,
    네트워크형 정보검색은 별도 비동기 실행(필요 시 서브에이전트)으로 이어서 집계한다.
